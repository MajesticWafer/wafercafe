What are mirrors?

Mirrors are copies of SM64-JS, Most schools have this website blocked so we are giving you other websites that arent blocked.

How to create a mirror?

Download the code for this REPL and host it on your website. Your mirror can be added to the list but your REPL won't be added to the list if:
It is hosted on a code sharing platform
It is hosted on a freenom domain (tk, ml, ga, cf, gq) as it could redirect to spam or it would shut down if it gets popular.
The mirror was made via a local tunnel software (ngrok, loca.lt, etc)
Once you have made the mirror and followed the Guidelines above then we will add you to the list.
