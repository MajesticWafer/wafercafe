Currently, only normal joypads are supported, and only controller 1 has controls set up.

| Button | Key         |
| ------ | ----------- |
| Up     | Up arrow    |
| Down   | Down arrow  |
| Left   | Left arrow  |
| Right  | Right arrow |
| Start  | Enter       |
| Select | Right shift |
| A      | X           |
| B      | Z           |
| X      | S           |
| Y      | A           |
| L      | D           |
| R      | C           |

Additionally, the following command are available:

| Key | Action            |
| --- | ----------------- |
| R   | Soft reset        |
| E   | Hard reset        |
| P   | Pause             |
| O   | Frame advance     |
| T   | Turbo (hold)      |
| L   | Run one CPU cycle |
| K   | Run one SPC cycle |
| J   | Dumps some data   |
| M   | Make save state   |
| N   | Load save state   |

Alt+Enter can be used to toggle fullscreen mode.
